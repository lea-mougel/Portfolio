Rubrique 4 : Projet Final CRM

Contexte : Conception et developpement d'une application CRM fullstack pour gerer un cycle commercial complet: contacts, entreprises, leads, pipeline, taches et communications. Entreprise prise en exemple : Dassault Systèmes

Stack Technique

  • Frontend : Next.js (App Router), TypeScript, Tailwind CSS.
  • Backend : API REST NestJS + TypeScript.
  • Auth & Donnees : Supabase Auth + PostgreSQL.
  • Emails : Brevo pour l'envoi et l'historisation des communications.
  • Roles : admin, commercial, user (droits filtres cote frontend/backend).

Fonctionnalites techniques realisees

  • Authentification complete (signup/login/logout) via Supabase.
  • CRUD contacts, entreprises et leads avec recherche/filtrage.
  • Pipeline de vente en vue colonnes et tableau + insights.
  • Module admin pour suivi des commerciaux.
  • Gestion des taches liees aux contacts/leads.
  • Module communications avec envoi email Brevo.

Livrables du Projet

Le projet inclut la documentation technique frontend/backend, la modelisation des donnees (MCD/ER), les cas d'usage et le code source.
